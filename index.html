Features

Backup multiple folders to multiple destinations

Each backer configuration is treated independently. Each folder can be backed up to different git repository destination.

Backups run in parallel in the background effortlessly.

Simplified SSH Authentication

Simply configure your SSH key path for the repository. If you are backing up multiple folders to multiple remote destinations, you can configure different SSH keys.

Leverages File system notifications

Efficiently identifies file changes using file system notifications instead of polling. This goes nice on your CPU.

Start/Stop an individual backup as you wish

You can still maintain a list of backup configurations and run the one you need and stop the rest.

Batteries included like Edit, Duplicate and Delete a configuration quickly.

Product Use Cases

Backup your offline notes from your favourite apps

Offline people love offline note taking apps like obsidian. You can use Git backer to automatically backup your obsidian vaults to your private git repositories for safe keeping.

Backup screenshots/mindmaps

When you are working on a project or any idea, Often you would like to take screenshots or create a mindmap of each and every step. Back that up to your git repository.

Offline password managers

When you would like to backup your encrypted password manager database file to a private repository. That can be backed up too.
